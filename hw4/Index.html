Main characteristics of PWA:

  • Full rsponsiveness and browser compatability - compatable with all devides, regardless of screen size and spedification
  • Connectivity independence - PWA can work both offline and on low-quality networks
  • Push Notifications - Can push notifications to alert user
  • Self-updates - Able to update themselves automatically
  • Safety - Apps are served through HTTPS
  • Discoverability and easy installation - Distributed through URL
  • Web App manifest - JSON file that gives PWA an app like interface
  • Service worker - offline work mode, background syncs, and push notifications
